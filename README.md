# Fish-Mail
The mod mail bot for Fish Hosting

### How to set up
Make a .env file, Just like this

```
DISCORD_BOT_SECRET=Token here
DATABASE_LINK=Mongo link
```

You can get a mongo link [here](https://www.mongodb.com/)

## Commands
```
<close
<reply <message>
<areply <message>
<block <user>
<unblock <user>
```
